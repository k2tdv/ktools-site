What KTools2 does

Automation tools built for real Revit workflows.

🤖

AI Assistant

Ask questions and get Revit help using Gemini AI — directly inside Revit.

📄

Bulk Sheet Creation

Create dozens of sheets and views at once. No more doing them one by one.

🖼️

Viewport Tools

Adjust and align viewports across multiple sheets quickly and accurately.

🔗

MEP Connections

Connect, move, and rotate MEP elements by connector — precisely and fast.

✏️

Annotation Copy

Copy annotations between views while keeping their positions correct.

🔒

Secure License

Machine-locked, cryptographically signed license. Offline and online support.
